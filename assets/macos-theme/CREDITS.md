# macOS Theme Wallpaper Credits

The rotating desktop and lockscreen wallpapers in `assets/macos-theme/landscapes/`
come from the Catppuccin wallpaper collection by zhichaoh.

- Source repository: [zhichaoh/catppuccin-wallpapers](https://github.com/zhichaoh/catppuccin-wallpapers)
- Source directory: [landscapes](https://github.com/zhichaoh/catppuccin-wallpapers/tree/main/landscapes)
- License: [MIT License](https://github.com/zhichaoh/catppuccin-wallpapers/blob/main/LICENSE)
- Local behavior: one landscape is selected per local calendar day and applied to both the desktop and lockscreen.
